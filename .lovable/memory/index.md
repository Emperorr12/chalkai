# Project Memory

## Core
Chalk - AI visual learning companion with Mr. White professor character.
Chalk-white #FAFAF7 bg, chalk-blue #3B6FCA accent, warm cream whiteboard, Caveat font for handwriting.
Inter 300/400/500 only. Line-height 1.75. No dark mode. No gradients.
Mr. White is a 140px chalk-drawn SVG professor with 7 animation states.
